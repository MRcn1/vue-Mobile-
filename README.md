# 开始

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli)

``` bash
# 安转依赖
npm install

# 启动服务器
npm run dev

# 生产
npm run build

```

# Folder structure
* build - webpack config 文件
* config - webpack config 文件
* dist - build 之后的文件
* src - 文件夹
    * assets 静态文件
    * pages - 页面 
    * store - vuex
    * App.vue
    * main.js - 入口文件
    * routes - 路由配置表
* static - 静态文件
